医学の三つの根拠―直感派・メカニズム派・数量化派
日本では医学的根拠の混乱が続いている。そのため多くの公害事件や薬害事件などで被害が拡大した。混乱の元は、医師としての個人的な経験を重視する直感派医師と、生物学的研究を重視するメカニズム派医師である。臨床データの統計学的分析（疫学）という世界的に確立した方法が、なぜ日本では広まらないのか。医学専門家のあり方を問う。
